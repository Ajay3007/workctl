## Q.11 — Optimization: Set of Minimizers [1 Mark]
$$f(x,y) = (x+y-1)^2 + (x+y)^2$$

**Answer: (B) Infinite elements**

### Step-by-Step:

**Step 1:** Let substitution s = x + y:
$$f = (s-1)^2 + s^2 = s^2 - 2s + 1 + s^2 = 2s^2 - 2s + 1$$

**Step 2:** Minimize over s:
$$\frac{df}{ds} = 4s - 2 = 0 \implies s = \frac{1}{2}$$

**Step 3:** The minimum occurs when x + y = 1/2.

This is a **straight line** in ℝ², containing **infinitely many** points (x, y) = (t, 1/2 − t) for any t ∈ ℝ.

**Minimum value:** f = 2(1/2)² − 2(1/2) + 1 = 1/2 − 1 + 1 = **1/2**

## Q.14 — Matrix Power [1 Mark]
$$P = \begin{bmatrix}2&1&0\\-1&0&0\\0&0&1\end{bmatrix}$$

**Answer: (A) P² = 2P − I**

### Step-by-Step:

**Step 1:** Find characteristic polynomial of the 2×2 top-left block:
$$P_{2\times2} = \begin{bmatrix}2&1\\-1&0\end{bmatrix}$$
$$\lambda^2 - \text{tr}\cdot\lambda + \det = \lambda^2 - 2\lambda + (0+1) = \lambda^2 - 2\lambda + 1$$

**Step 2:** By Cayley-Hamilton theorem: P² − 2P + I = 0 (for the 2×2 block)
$$\therefore P_{2\times2}^2 = 2P_{2\times2} - I$$

**Step 3:** The third row/column is just [0,0,1], so P³₃ = 1 also satisfies P² = 2P − I (since 1 = 2(1) − 1).

**Verify directly:**
$$P^2 = \begin{bmatrix}2&1&0\\-1&0&0\\0&0&1\end{bmatrix}\begin{bmatrix}2&1&0\\-1&0&0\\0&0&1\end{bmatrix} = \begin{bmatrix}3&2&0\\-2&-1&0\\0&0&1\end{bmatrix}$$

$$2P - I = 2\begin{bmatrix}2&1&0\\-1&0&0\\0&0&1\end{bmatrix} - \begin{bmatrix}1&0&0\\0&1&0\\0&0&1\end{bmatrix} = \begin{bmatrix}3&2&0\\-2&-1&0\\0&0&1\end{bmatrix} \checkmark$$

## Q.15 — Conditional Expectation [1 Mark]
**Given X = 1, find E[Y|X=1]**

**Answer: (B) 1/2**

### Step-by-Step:

**Step 1:** Find P(X=1):
$$P(X=1) = P(X=1, Y=0) + P(X=1, Y=1) = \frac{1}{8} + \frac{1}{8} = \frac{1}{4}$$

**Step 2:** Find conditional probabilities:
$$P(Y=0 | X=1) = \frac{P(X=1, Y=0)}{P(X=1)} = \frac{1/8}{1/4} = \frac{1}{2}$$
$$P(Y=1 | X=1) = \frac{P(X=1, Y=1)}{P(X=1)} = \frac{1/8}{1/4} = \frac{1}{2}$$

**Step 3:** Compute E[Y|X=1]:
$$E[Y|X=1] = 0 \cdot P(Y=0|X=1) + 1 \cdot P(Y=1|X=1) = 0 \cdot \frac{1}{2} + 1 \cdot \frac{1}{2} = \boxed{\frac{1}{2}}$$

## Q.27 — Power Factor Calculation [1 Mark]

## Q.65 — PD Controller Design for Phase Margin [2 Marks]
$$G(s) = \frac{1000\sqrt{2}}{s(s+10)^2}, \quad D(s) = 1 + K_D s$$
**Phase margin = 45° at gain crossover ω = 10 rad/s. Find KD.**

**Answer: KD = 0.2**

### Step-by-Step:

**Step 1:** Compute open-loop TF at ω = 10:
$$L(j\omega) = D(j\omega) \cdot G(j\omega) = (1 + jK_D\omega) \cdot \frac{1000\sqrt{2}}{j\omega(j\omega+10)^2}$$

**Step 2:** Gain crossover condition |L(j10)| = 1.

First compute G(j10):
$$G(j10) = \frac{1000\sqrt{2}}{j10 \cdot (j10+10)^2}$$

$$(j10+10)^2 = (10+j10)^2 = 100(1+j)^2 = 100(1+2j-1) = 200j$$

$$G(j10) = \frac{1000\sqrt{2}}{j10 \cdot 200j} = \frac{1000\sqrt{2}}{2000j^2} = \frac{1000\sqrt{2}}{-2000} = -\frac{\sqrt{2}}{2} = -\frac{1}{\sqrt{2}}$$

**Step 3:** Phase of G(j10):
$$\angle G(j10) = -90° - 2\times 45° = -180°$$

(Each (j10+10) contributes 45° phase lag, and j10 contributes 90° lag)

**Step 4:** Phase margin condition:
$$PM = 180° + \angle L(j\omega_{gc}) = 45°$$
$$\angle L(j10) = 45° - 180° = -135°$$

**Step 5:** Phase of L(j10) = phase of D(j10) + phase of G(j10):
$$-135° = \angle(1 + j10K_D) + (-180°)$$
$$\angle(1 + j10K_D) = -135° + 180° = 45°$$

$$\tan(45°) = \frac{10K_D}{1} = 1$$
$$10K_D = 1 \implies K_D = \boxed{0.1}$$

**Step 6:** Verify gain crossover with KD = 0.1:
$$D(j10) = 1 + j10(0.1) = 1 + j = \sqrt{2}\angle 45°$$

$$|L(j10)| = |D(j10)| \cdot |G(j10)| = \sqrt{2} \times \frac{1}{\sqrt{2}} = 1 \checkmark$$

**Answer: KD = 0.1**

## Q.2A — Z-Transform [2 Marks]
$$X(z) = \frac{(1-a^2)}{(z-a)(z^{-1}-a)}, \quad a < |z| < 1/a$$

- Pole at z=a (inside ROC outer boundary) → causal term aⁿu[n]

For n ≥ 0, the answer is **x[n] = aⁿu[n]** → **(A)**
